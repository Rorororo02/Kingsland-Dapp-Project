Decentralized Mobile Number Registry

Welcome to the Decentralized Mobile Number Registry! This distributed app runs on the Ethereum blockchain network and holds a registry of mobile numbers as Solidity smart contract.
  • Contract owner can register mobile number to be stored on the blockchain.
  • Users can view the existence of a mobile number!
  • This application will help you avoid frauds and scams!

Register Mobile Number

Contract owners can register (upload) new mobile number to the "Mobile Number Registry" smart contract on the Ethereum blockchain decentralized network.

Verify Mobile Number

Blockchain users can verify mobile number on the Ethereum network. Know who is attempting to scam you!
